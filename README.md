## Instalacion ⚒️

1. Abre la terminal de comandos en donde se encuentre la carpeta del programa
2. Utiliza los siguientes comandos en orden para poder instalar el sistema

```bash
composer install
```

```bash
php artisan key:generate
```

```bash
npm install
```

```bash
npm install yarn
```

```bash
yarn run dev
```

```bash
php artisan serve
```

```bash
yarn prod
```
